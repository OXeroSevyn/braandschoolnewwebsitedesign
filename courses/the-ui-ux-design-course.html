Interactive Design Track

Master the Art of UI/UX.

Course Roadmap.

Module 01

UI/UX Intro & Figma Basics

  • Understanding user interface vs user experience design principles
  • Touring the Figma interface, layouts, layers, and artboards
  • Creating simple UI designs using vector shapes and typography
  • Hands-on assignment building your first digital wireframe card
Module 02

User Research & Prototyping

  • Learning UX research strategies, user personas, and journey mapping
  • Introduction to interactive prototyping in Figma (links and frames)
  • Complementing designs with Framer for advanced interactivity
  • Assignment: Designing interactive wireframe flows from scratch
Module 03

Figma Tools & Design Principles

  • Mastering vector editing, component styling, and auto layout
  • Typography hierarchies and color theory applications for UI layouts
  • Structuring shared UI components and basic design libraries
  • Creating mid-fidelity templates with strict grid frameworks
Module 04

User-Centric Design & Feedback

  • Reviewing UI design best practices and responsive layout rules
  • Conducting live usability testing and mapping user feedback
  • Refining components and interface pages based on testing cycles
  • Structuring multi-resolution screens for mobile, tablet, and desktop
Module 05

Advanced Prototyping & Motion

  • Creating nested animations, overlays, and smart-animate flows
  • Applying micro-interactions to buttons, toggles, and menus
  • Enhancing high-fidelity screens with code-based Framer animations
  • Structuring dynamic scroll behaviors and parallax effects
Module 06

Developer Collaboration & Specs

  • Version control, live commenting, and team design collaboration
  • Preparing handoff documentation, generating CSS specs, and exporting assets
  • Managing typography scales and design system tokens
  • Collaborating with developers to review mockups for deployment
Module 07

Specialization Workshops

  • Mobile App Design workshops (native iOS/Android patterns)
  • Responsive Web Design and e-commerce checkout flows
  • Designing high-converting landing pages for businesses
  • Kicking off final case studies and client projects
Module 08

Final Project & Presentation

  • Guide reviews, one-on-one instructor feedback and refinements
  • Preparing a comprehensive case study presentation for your portfolio
  • Final cohort presentations to design experts and class alumni
  • Completing developer-ready design links and high-fidelity mockups
